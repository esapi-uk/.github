# UK ESAPI Community

This GitHub organisation hosts documentation and example governance material shared by members of the UK ESAPI community.

The aim is to help centres developing Eclipse Scripting (ESAPI) tools share
approaches to documentation, testing, and governance of clinical scripting
projects.

## What is shared here

Repositories may contain examples such as:

- Script specification templates  
- Validation and testing checklists  
- Risk assessment or hazard log templates  
- Deployment or version control procedures  
- Governance documentation for ESAPI development  
- Workshop materials or presentations  

---

## What should NOT be uploaded

This space is for documentation only. Please **do not upload**:

- ESAPI source code  
- Compiled binaries (`.dll`, `.exe`)  
- Patient data or identifiers  
- API keys, credentials, or configuration secrets  
- Server names, database details, or internal system identifiers  
- Network diagrams or infrastructure architecture  

Please ensure all uploaded documents are scrubbed of sensitive information,
including patient identifiers, internal hostnames, or other security-sensitive
details.

---

## Status of documents

Documents shared in this organisation are examples of current practice within
UK radiotherapy centres. They are provided for information and discussion only.

They are **not validated clinical procedures** and must not be adopted without
appropriate local governance, verification, and risk assessment.

## Contributing

Community members are welcome to contribute documentation examples.

Typical contributions include:

- Templates used in local ESAPI development  
- Testing or validation approaches  
- Governance documentation  
- Workshop materials  

Please remove centre-specific identifiers where possible and ensure that
documents are suitable for public sharing.
